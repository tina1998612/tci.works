課程特色


課程並不是使用任何台灣傳統的教材，那樣不但侷限住你的眼界，不直覺的教學方式甚至讓你學不懂， 更怕學了已經被淘汰的知識。
請放心，我的背景使我不同，追求的是「顛覆」的教學方式，最「新」的知識，最「有用」的內容。

1. 學會寫程式關鍵基礎

用最輕鬆的方式學會寫程式是課程的首要任務。程式語言有上百種，但關鍵基礎是完全互通的，學會這些基礎不管走什麼領域，都極為重要。

New

2. 隨時發問討論

線上課程最擔心的就是無法問問題，但參與過線上討論的同學都知道，真的不會無法發問。集中討論對大家幫助很大，這也是美國電腦科學系實際使用的發問平台。

3. 透過實作學習

透過實作是最好的學習方式，教學方式不是死板的介紹程式章節跟專有名詞，而是一步一步帶著你把程式做出來，一行一行的讓你完全聽懂。

New

4. 結合美國頂尖大學的教材

為了使用最顛覆，最能輕鬆學會的方式教學寫程式，我整合多所美國頂尖大學實際使用的教材，與各大知名網站的實作練習，為了讓你學到最新、最有用的知識。

5. 打造程式履歷表

除了學會寫程式，更讓你在學習的過程中把程式透過GitHub進行版本控管，在GitHub網站上累積個人程式，這已經是現代科技業的履歷表，極為重要。

New

6. 了解現代科技領域與趨勢

不是只是制式的教學【寫】程式，而是在過程中，打開你的眼界，了解程式有多強大，為什麼所有領域皆仰賴程式，包括最夯領域：人工智慧

常見問題

★ 在哪裡上課？
目前課程都是『線上課程』，所以課程上線後，隨時隨地都可以透過手機、平板、與電腦在上看課程影片，沒有時間和地點的問題！
★ 課程可以看幾次？
課程可以永久 無限次數重複觀看。
★ 我還是學生可以報名嗎？
歡迎有興趣且時間能配合的同學報名參加，課程沒有限制年齡和學歷。
★ 一周的課程是分成幾次上課，有分線上線下，或是都是預錄好的影片呢？如果課程期間需要進行補課該怎麼辦？
每週的課程都會在線上進行，所以有需要請假的同學可以利用影片補課或是複習不清楚的地方，請不用擔心。 唯有考試會是線下進行，所以考試時間切勿安排其他事務，以免影響個人權益。
★ 完全沒有基礎可以學習嗎
沒問題，課程就是適合任何零基礎想上手區塊鏈的同學。
電腦需要什麼設備？
線上課程可以用任何平台播放，寫程式則不行使用手機或平板來寫，必須使用電腦。 一般電腦即可，不需要任何特殊的電腦或是特別高階的電腦。
能每週要花多久的時間學習，才能順利結業，課程的難易度，及需使用的程式語言？
我們的訓練過程是六週，學習時間依每人學習狀況和背景經驗 每週需要至少 20 小時。我們都有妥善的安排進度，若是跟不上可以多看幾次課程來補足。 程式語言部分智能合約會使用 Solidity，前後端則是 Javascript 和 Nodejs，這部分個人需要先有基礎能力，我們會直接從區塊鏈相關的技術講起。
有線上進階課程嗎？
Hahow是基礎課程，提供零基礎及不同領域的同學學習。每季展開的菁英招募計畫就是進階課程，針對四類人才菁英培訓對接企業。
上課時遇到問題怎麼辦？
有線上討論區讓同學發問/討論，老師會迅速於線上討論區回答，也會進行每週直撥現場解答。

企業顧問

無法否認，寫程式是現代世界最重要的能力。

台灣教育體系失敗，學歷氾濫，許多學生僅追求文憑，甚至選校不選系， 完全對程式教育並沒有理解其重要性。每分每秒，當美國在開發人工智慧，無人商店，為什麼台灣的學生還在熬夜死背數學公式

一定要理解 學什麼才【有用】

根據統計，美國前50%高薪職位，全部需求這個技能-寫程式。大學畢業生寫程式平均入行薪水為年薪180萬台幣。許多先進國家早已把寫程式納入國小國中課程，台灣卻還是不為所動。 在美國學生搶破頭想要申請的系，在台灣卻是許多人根本沒聽過電腦科學系。在美國申請電腦科學的人數是電機系的6倍左右，工作職缺甚至贏出十倍的規模。當你跨越傳送門，到達了加州矽谷科技公司，此時你會深刻體會什麼才是最重要的技能。

打開國際觀，擺脫傳統教育束縛

物聯網，無人車，虛擬實境，人工智慧，世界改變的速度遠超乎你的想像。不管申請研究所與教授面試，還是工作面試，老闆第一句就問:"會寫程式嗎"，當下你才會恍然大悟，以前背誦的文言文與化學反應式，熬夜讀書多換來的5分10分，並沒有幫到你。 搭一趟13小時的飛機，過一個海關，你到了美國Google總部，台灣教育給你背誦的所有資訊，卻幫不上忙。 比起失敗的傳統課綱，背了就多分的考試制度，公司想要的是怎麼樣的人才，需要什麼樣的能力，才是你應該思想的重點。

課程延展性

  • 必學課程

    基礎程式課程

    由此開始讓你對科技大開眼界！除了教會你寫程式，也從根基讓你了解真正業界在使用的科技與開發方式，「好的基礎」很重要。由此帶你用最輕鬆的方式上手寫程式。

  • 建議課程

    進階程式課程

    進階課程一樣是以實作程式的方式帶領大家學習一些進階的程式概念，例如物件導向、製作程式圖像畫面等等，由於程式碼的結構複雜性稍微提高，建議適合國中以上。

  • 興趣課程

    資料科學/人工智慧

    學完基礎程式課程的同學，如果對數據分析有興趣，也可以不學進階課程直接切入資料科學/人工智慧領域。人工智慧其實所指的就是【機器學習】技術，這不但是數據分析頂尖的方式，也是目前科技界的最夯的領域。

  • 興趣課程

    金融市場 x 程式交易

    程式交易是近年來金融圈最熱門的領域，從實戰角度學習金融市場，再透過程式交易來達到安全、穩定的交易方式。
